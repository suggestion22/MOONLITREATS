# Security Policy

## Supported Versions

The `main` branch is the supported version of MOONLITREATS.

## Reporting a Vulnerability

Please report security issues privately through GitHub's security advisory flow when available. Do not disclose suspected vulnerabilities publicly until they have been reviewed.

For this static site, please include:

- The affected page or file
- Steps to reproduce
- Expected and actual behavior
- Browser and device details, when relevant

We will review reports and apply fixes to the `main` branch.
